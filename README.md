# Tic Tac Toe
## Swift 5 + UIKit

## How to run

- Clone the project
- Open with XCode (open tictactoe.xcodeproj)
- Be sure that you are on "tictactoe" target
- Choose on which device you want to run the application
- Click on the "Play" button


## How to run tests
